# APEX Digital Gallows

EU AI Act 2026 Compliance Verification Platform

## About

APEX Digital Gallows provides privacy-preserving compliance verification for the EU AI Act 2026 enforcement wave. Verify AI systems without disclosing sensitive details.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```sh
npm run build
```

## Deployment

Deploy to any static hosting (Vercel, Netlify, Cloudflare Pages, etc.)
